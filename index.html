Bienvenidos

¡Hola! Mi nombre es Jhojan Arango, soy desarrollador frontend de Cali, Colombia. Manejo tecnologías como HTML5, CSS3, JavaScript y jQuery. Aunque aún no tengo experiencia trabajando con empresas, estoy en constante aprendizaje y me apasiona crear proyectos web dinámicos y funcionales. Estoy dispuesto a seguir creciendo y aplicando mis habilidades en nuevos desafíos.

Mis proyectos

ToDo List

He creado una ToDo List utilizando HTML, CSS y JavaScript. Este proyecto permite a los usuarios agregar tareas, marcarlas como completadas y eliminarlas. La interfaz se actualiza de forma dinámica sin necesidad de recargar la página, lo que hace que la gestión de las tareas sea rápida y eficiente.

Calculadora

La calculadora que desarrollé realiza operaciones matemáticas básicas, como suma, resta, multiplicación y división. Utiliza HTML para la estructura, CSS para el diseño y JavaScript para ejecutar los cálculos y mostrar los resultados en tiempo real.
